Установка и обновление
======================

.. toctree::
   :maxdepth: 3
   :hidden:
   :caption: Содержание:

Установка с помощью `PyPi <https://pypi.org/project/oceanai/>`_
---------------------------------------------------------------

.. code-block:: sh

   pip install oceanai

Обновление с помощью `PyPi <https://pypi.org/project/oceanai/>`_
----------------------------------------------------------------

.. code-block:: sh

   pip install --upgrade oceanai
